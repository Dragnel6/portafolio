Innovation 3D

Es una app web desarrollada con el objetivo de brindar servicios de impresión 3D como creación de prototipos pequeñas y medianas tiradas de piezas con la función de una calculadora hecha para moldear tu diseño como gustes.

Para el desarrollo de esta pagina web se utilizo el lenguaje de programacion Python y Flask que es un framework escrito en python que permite crear aplicaciones web rapidamente con un minimo de lineas de codigo, se utilizo tambien el gestionador de base de datos MYSQL para el guardado de los diseños 3d. Para el desarrollo del diseño comunmente utilize HTML y boostrap con un toque de CSS.
